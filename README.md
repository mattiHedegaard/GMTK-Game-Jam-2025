# Matti Hedegaards participation in GMTK Game Jam 2025
This project were made to participate in Game Makers Toolkits 2025 Game Jam.
The Game Jams theme were "Loop", therefore this project ended up as trains going in loops, picking up passengers and driving them to their destinations to earn money.
The project were made single handed by me, Matii Hedegaard, and were made in 96 hours.
